Введение в HTML

HTML (HyperText Markup Language) - это фундаментальная технология, используемая для создания структуры веб-страницы и её контента. HTML не является языком программирования; это язык разметки, который определяет структуру вашего контента.

Почему важно изучать HTML?

HTML является основой веб-разработки. Даже если вы планируете использовать современные фреймворки и библиотеки, понимание HTML необходимо для создания качественных, доступных и хорошо структурированных веб-страниц.

Базовая структура HTML-документа

<!DOCTYPE html> <html lang="ru"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Название страницы</title> </head> <body> <h1>Мой первый заголовок</h1> <p>Мой первый параграф.</p> </body> </html>

Основы HTML

HTML состоит из серии элементов, которые вы используете, чтобы охватывать, оборачивать или размечать различные части контента, чтобы он появлялся определенным образом или действовал определенным образом.

Структура HTML-элемента

Основная структура HTML-элемента выглядит следующим образом:

<тег атрибут="значение">Содержимое</тег>

Например: <a href="https://example.com">Пример ссылки</a>

Блочные и строчные элементы

HTML-элементы можно разделить на две основные категории:

Блочные элементы

Занимают всю доступную ширину, всегда начинаются с новой строки. Примеры: <div>, <h1>-<h6>, <p>, <ul>, <li>.

Строчные элементы

Занимают только необходимую ширину, не начинаются с новой строки. Примеры: <span>, <a>, <img>, <strong>.

HTML-элементы

Заголовки

HTML предоставляет шесть уровней заголовков, от <h1> (самый важный) до <h6> (наименее важный).

<h1>Заголовок 1 уровня</h1> <h2>Заголовок 2 уровня</h2> <h3>Заголовок 3 уровня</h3> <h4>Заголовок 4 уровня</h4> <h5>Заголовок 5 уровня</h5> <h6>Заголовок 6 уровня</h6>

Ссылки

Ссылки создаются с помощью тега <a>. Обязательный атрибут href указывает адрес назначения ссылки.

<a href="https://www.example.com">Это ссылка на Example.com</a>

HTML-формы

Формы используются для сбора пользовательского ввода. Они содержат различные элементы ввода, такие как текстовые поля, флажки, переключатели, кнопки отправки и т.д.

Пример формы обратной связи

<form> <label for="name">Имя:</label> <input type="text" id="name" name="name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> <label for="message">Сообщение:</label> <textarea id="message" name="message"></textarea> <button type="submit">Отправить</button> </form>
